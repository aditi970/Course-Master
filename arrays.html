Arrays in C++


    It is a homogeneous collection of data.Here the data is placed in contiguous memory location that can be individually called out as and when needed with the help of loops and various our features.

    Advantages of Arrays:

  • Easy to specify.
  • Less coding is required because all the data is grouped.
  • High performance
  • Random accessing of data is possible.
  • Free from runtime overhead.

    • Disadvantages of Arrays:

  • It can store only same type of data.
  • We need to know its size.
  • Careful designing is required.

Types of Arrays:

  • Single Dimentional arrays
  • 2-Dimentionalarrays
  • Multi Dimentional arrays

Operations that can be performed on arrays:

  • Searching
  • Sorting
  • Merging of 2 arrays
  • Insertion in an array
  • Delation from an array
